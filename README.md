# MHRNGenerator

## Description
- **Memory**: Генерация ландшафта происходит посредством последовательной зависимости между каждым уровнем (x,y).
- **Hill**: При генерации образуются холмоподобные структуры.
- **River**: При генерации так же образуются вытянутые структуры в форме дельт рек.
- **Noise**: В ходе генерации образуется шум для близости к реальному ландшафту.
 
## Algorithm

* **Анализ медианы (Seeding):** На основе медианного значения первой строки определяются начальные активные зоны (будущие хребты или русла).
* **Трансформация (Range Evolution):** Активные зоны динамически сужаются или расширяются на основе рандомной генерации и сравнения со средним значением высоты в текущем сегменте.
* **Смещение (Offset):** Каждая активная зона на каждом шаге случайно сдвигается влево, вправо или остается на месте, создавая естественную извилистость структур.
* **Инерция высот (Lerp):** Новые значения пикселей вычисляются через линейную интерполяцию относительно предыдущего ряда, что обеспечивает плавные переходы и исключает резкие обрывы.
* **Фоновое заполнение:** Перед отрисовкой активных зон весь ряд проходит через стадию «затухания» к базовому уровню, формируя мягкие подножия и чистые границы объектов.
* **Жизненный цикл:** Если в ходе сужения диапазон индексов становится пустым, структура прекращает свое существование, имитируя завершение хребта или реки.
